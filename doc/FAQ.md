# 常见问题

> [English Version](FAQ_EN.md)

## 连接不上 DG-LAB App？

- 确保手机和Frisson在同一局域网中
- 检查Frisson端口6969是否被防火墙拦截（安装包会自动配置防火墙规则）

## 标准版和自包含版有什么区别？

- **标准版**：需要安装 .NET 10运行时，安装包体积小
- **自包含版**：无需额外运行时，开箱即用，但安装包更大

## 外部程序如何通过 JSON 协议控制？

外部程序通过WebSocket连接Frisson，发送特定格式的JSON消息来控制设备强度等参数。可参考 `demo/` 目录下的示例脚本 `tetris.py`。

## 连接后无法调节强度？

- 确认设备已通过DG-LAB App成功配对。
- Frisson允许用户设置与DG-Lab App所设不同的强度上限。当新设置的强度大于DG-Lab App的上限时，它会忽略该控制消息，致使无法调节强度。
>可在设置->执行器页面勾选“使用执行器上限”以保持上限同步

## 支持哪些设备？

目前支持DG-LAB郊狼3.0设备，通过DG-LAB App建立WebSocket连接。

## 如何查看通信日志？

应用内设置->关于页面可以查看WebSocket通信日志。
