# Frisson

>[简体中文](../README.md)

Frisson connects to the Coyote 3.0 device, control device strength and limits, while also supporting remote control from external programs via a JSON protocol.
 
## Features

- **Connect to DG-LAB devices** — Establish connection with DG-LAB App via WebSocket
- **Strength and limit control** — Adjust and control A/B channel intensity and limits
- **Remote control from external programs** — Control remotely from external programs via a JSON protocol

## Download

Download the latest installer from [Releases](https://github.com/curtainsmall/frisson/releases).

| Variant | Description | System Requirements |
|---------|-------------|---------------------|
| `Frisson-Setup-X.Y.Z.exe` | Standard | Windows 10/11 (64-bit), requires [.NET 10 Runtime](https://dotnet.microsoft.com/download) |
| `Frisson-Setup-X.Y.Z-SelfContained.exe` | Self-contained | Windows 10/11 (64-bit), no additional installation needed |
